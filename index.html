Course

About the course

Generative AI for Work & Research is a hands‑on, guided program covering websites & documents, AI‑assisted data analysis, text analysis for research, custom chatbots, task automation, and communication & outreach — culminating in a capstone showcase.

Weekly live workshops
Live help sessions
Private community forums
Personalized feedback
Capstone coaching
Shareable certificate
Module 1 — Websites, Documents & Presentations
Learn how LLMs work and generate high‑quality text for papers or grants; build simple research websites; structure documents with Markdown; and design clear, AI‑assisted presentations.
LLM overview Markdown / LaTeX / HTML No‑code sites
Module 2 — AI‑Automated Data Analysis
Use tools like Code Interpreter and Julius to process datasets, visualize results, and create automated reports and dashboards.
Code Interpreter Dashboards (TheBricks) Batch file transforms
Module 3 — GenAI for Research & Text Analysis
Techniques for literature search, text classification, sentiment analysis, knowledge maps, and automated summarization.
Elicit / LiteRev Text classification Summarization pipelines
Module 4 — Custom GPTs & Chatbots
Build research assistants with RAG, Botpress, and knowledge base integration; understand context windows and chat architectures.
RAG systems Botpress Knowledge bases
Module 5 — AI for Task Automation
Practical automations: email response flows, browser control agents, and agent‑based systems for recurring tasks.
Email automations Agents for monitoring Browser control
Module 6 — Communication & Outreach
Create summaries for different audiences, scientific figures, and video abstracts; schedule content and refine prompts for visualization.
Research summaries Figures & diagrams Video abstracts
Module 7 — Capstone Project & Showcase
Build and present a final AI solution or tool, with feedback, peer review, and a public Q&A.
Peer feedback Public presentation
